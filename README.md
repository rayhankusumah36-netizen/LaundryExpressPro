# LaundryExpress Pro

LaundryExpress Pro adalah program Sistem Manajemen Laundry Digital berbasis bahasa pemrograman C++ yang dirancang untuk membantu pengelolaan usaha laundry modern, mulai dari pencatatan order, perhitungan biaya, hingga laporan pendapatan.

Program ini dibuat untuk memenuhi Tugas Proyek Mandiri dengan penerapan berbagai konsep dasar pemrograman C++.
